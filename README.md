Web and Game Developer
----------------------

Merhaba! Ben İsa, Bilgisayar Mühendisliği mezunu ve yazılım geliştirmeye tutkuyla bağlı biriyim. Özellikle web geliştirme, Unity ile oyun geliştirme ve gömülü sistemler (Arduino, IoT) alanlarına ilgi duyuyorum. Öğrenmeye açık, yeni teknolojileri keşfetmekten keyif alan ve sürekli kendini geliştirmeyi hedefleyen biriyim.

Burada hem kendi projelerimi paylaşıyor hem de farklı teknolojilerle denemeler yaparak deneyimlerimi artırmaya çalışıyorum. Özellikle ASP.NET Core, React, C#, JavaScript gibi teknolojilerle çalışmalar yapıyor, aynı zamanda yeni teknolojiler konularında kendimi geliştirmeye devam ediyorum.

Amacım, gerçek hayatta kullanılabilir, ölçeklenebilir ve kullanıcı dostu projeler geliştirmek.
